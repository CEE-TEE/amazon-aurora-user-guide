# Aurora PostgreSQL functions reference<a name="Appendix.AuroraPostgreSQL.Functions"></a>

Following, you can find a list of Aurora PostgreSQL functions that are available for your Aurora DB clusters that run the Aurora PostgreSQL\-Compatible Edition DB engine\. These Aurora PostgreSQL functions are in addition to the standard PostgreSQL functions\.
